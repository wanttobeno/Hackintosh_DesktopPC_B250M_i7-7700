##### 正常使用

```
1、集显输出，使用外接显示器HDMI显示。

2、主机前后音频插口正常。

3、显示器HDMI接口支持音频输出，不支持音量调节。

4、休眠，唤醒，关机正常。

5、流畅，不卡屏、闪屏
```



##### 使用说明

复制efi，集显HDMI插口，安装系统。
